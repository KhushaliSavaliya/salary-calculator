# 💰 SalaryPro - Smart Salary Calculator

A beautiful, responsive salary calculator built with **HTML5, Tailwind CSS, and jQuery**. 

### 🚀 Features
- **Smart Calendar Logic**: Automatically detects total days in a month.
- **Custom Weekend Rules**: Automatically excludes all **Sundays** and the **2nd & 4th Saturdays**.
- **Leave Management**: Account for Paid Leaves (PL), Unpaid Leaves (LWP), and Public Holidays.
- **Fully Responsive**: Works perfectly on Mobile, Tablet, and Desktop.
- **Modern UI**: Clean design featuring Glassmorphism and smooth transitions.
